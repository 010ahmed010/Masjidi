

## Admin Dashboard

Intro: This page is the main admin control center for Masjidi. It manages public site content, teacher assignments, student records, attendance monitoring, honor approvals, and school settings. Admin actions directly affect the public HomePage and complete the teacher workflow.

1. Dashboard Home: A summary page with key metrics and quick links.
   - Total students
   - Total teachers
   - Total classes/courses
   - Pending honor approvals
   - Active occasions
   - Recent news items
   - Student registration status (open/closed)

2. Students Management Page: Add and manage student records.
   - Add new students (admin-only)
   - Edit student profiles and details
   - Assign students to classes and teachers
   - Search and filter by class, teacher, name, or status

3. Teachers Management Page: Manage teacher accounts and assignments.
   - Add new teacher accounts
   - Edit teacher information
   - Assign teachers to classes
   - View teacher class load and contact details

4. Classes / Courses Page: Manage classes and course listings.
   - Create and edit classes and course names
   - Set course images for HomePage display
   - Assign teachers to classes
   - Link classes to available courses shown on the public HomePage

5. Honor Control Page: Approve teacher nominations for the honor wall.
   - Review weekly honor nominations from teachers
   - Approve or reject each nomination
   - Add or edit honor descriptions for public display
   - Publish approved students to the HomePage Honor section

6. Occasions Management Page: Manage special occasion content.
   - Add or edit current occasion cards
   - Upload images for events like Ramadan and Eid
   - Enable or disable occasion display on HomePage

7. News / Offers Page: Manage public announcements and posts.
   - Create, edit, and delete news items
   - Publish offers and course announcements
   - Control which items appear in the HomePage News section

8. Attendance Monitoring Page: Monitor teacher-submitted attendance.
   - View recent attendance records by class and date
   - Monitor the most recent day shown on HomePage
   - Search/filter attendance data by class, student, or date
   - Optionally override or correct submitted records if needed

9. Contact & Site Info Page: Manage public contact details.
   - Update WhatsApp, phone, and email information
   - Set the masjid main picture
   - Configure Google Maps iframe location details
   - Edit the contact page short description and footer info

10. Certificates Page: Manage student certificates and releases.
   - Issue certificates to students
   - Track generated or released certificates
   - Manage certificate status and distribution notes

11. Global Settings Page: Control site-wide admin settings.
   - Toggle student registration open/closed
   - Update global header/footer settings
   - Manage admin preferences and site behavior

Navigation suggestion:
- Dashboard
- Students
- Teachers
- Classes / Courses
- Honors
- Occasions
- News
- Attendance
- Contact Info
- Certificates
- Settings

This structure matches the public pages and teacher workflows from the existing Masjidi description files and ensures the admin can fully manage the site.
